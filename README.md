# ポートフォリオサイト：技術的証明と制作実績

本サイトは、Webコーダーとしての実装スキルと、保守性の高い設計能力を証明するために制作されたポートフォリオサイトです。

## プロジェクトのコンセプト
単なる作品集としてだけでなく、実務における大規模な開発現場を想定した設計を行っています。
誰がコードを見ても理解しやすく、後からの修正や機能追加が容易な「高品質なコーディング」を体現することを目的としています。

## サイトの主な機能と特徴

### 1. 制作実績の検索・フィルタリング
過去の制作実績を、カテゴリーや使用技術で絞り込んで閲覧できます。ブラウザの「戻る・進む」ボタンに対応しており、検索結果の状態を維持したままスムーズに回遊できる、使い心地を重視した設計にしています。

### 2. スキルと実績の連動
自身のスキルセットを単に並べるだけでなく、それぞれの技術がどの実績で使用されているかを動的に紐付けています。これにより、スキルの習熟度を客観的な証拠とともに確認いただけます。

### 3. セキュリティに配慮したお問い合わせフォーム
スパム対策と確実なメール配信システムを組み込み、信頼性の高い連絡手段を構築しています。

### 4. 守秘義務（NDA）への対応
実務案件において公開制限がある実績については、システム側で自動的に画像をマスキング処理する機能を搭載しています。エンジニアとして情報の取り扱いに細心の注意を払っていることを示しています。

### 5. UIパーツギャラリー（準備中）
汎用的なUIパーツを一覧できる専用エリアを構築済みです。現在は先行公開のため「Coming Soon」としていますが、内部的にはサイト本体の影響を受けずにパーツ単体を検証できる高度なプレビュー環境を実装しています。

## 開発の品質について

### 読みやすさと保守性
プログラムの各部品を適切な場所に配置する設計思想を取り入れています。これにより、長期的な運用においてもコードが複雑化しにくい構造を実現しています。

### あらゆる端末への最適化
パソコン、タブレット、スマートフォンなど、どのデバイスで見ても最適なレイアウトで表示されるレスポンシブ設計を徹底しています。また、ブラウザの文字サイズ設定を尊重する単位指定を行うなど、アクセシビリティにも配慮しています。

### 検索エンジンへの露出制御
本サイトは採用担当者の方へ直接共有することを前提としているため、不特定多数に検索されないよう、検索エンジンにインデックスされない設定をシステム側で施しています。

## 使用技術
- 開発フレームワーク：React 19 / Next.js
- 言語：TypeScript
- スタイリング：Tailwind CSS / SCSS Modules
- アニメーション：GSAP
- フォーム・メール送信：Server Actions / Resend / Cloudflare Turnstile

## 運用について
新しい制作実績やスキルの更新は、専用のデータファイルを編集するだけでサイト全体に自動反映される仕組みを構築しており、迅速な情報更新が可能です。
