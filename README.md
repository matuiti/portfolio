# ポートフォリオサイト：実績と技術の証明

本サイトは、リモート環境下においても出来る限り実装力をお伝えするために開発したポートフォリオサイトです。

## プロジェクトのコンセプト
単なる作品集としてだけでなく、実務規模を想定した設計・ユーザー目線を加味したUX実装を行いました。

設計方針として、「Single Source of Truth (SSoT)」の徹底、コンポーネントの「Colocation（近接配置）」による認知負荷の低減により、保守性を担保します。
どこで何をやっているかが予測しやすいルールを一貫したコードベース設計と、後の修正や機能追加が容易になるように疎結合なモジュール分割を徹底しています。

## サイトの主な機能と特徴

### 1. 実績の検索・フィルタリング
制作実績を多様な方法で絞り込んで閲覧できます。ブラウザの「戻る・進む」ボタンに対応しており、検索結果の状態を維持したままスムーズに回遊できる、使い心地を重視した設計にしています。

### 2. スキルと実績の連動
スキルセットを単に並べるだけでなく、それぞれの技術と実績を動的に紐付けています。これにより、スキルの習熟度を客観的なエビデンスとともに確認いただく狙いを持って設計しました。

### 3. お問い合わせフォームとメールシステム連携
ユーザーの手間を最小化したスパム対策やフロントエンド・バックエンド両側から型安全を担保した多重バリデーションチェックによるセキュリティの考慮。
確実に到達できるようにメール配信システムを組み込み、信頼性の高い連絡手段を構築しています。なお、気軽な連絡手段を選択可能にする目的で直接のメール送信に対応。

### 4. 守秘義務（NDA）への対応
実務案件において公開制限がある実績については、システム側で自動的に画像をマスキング処理する機能を搭載しています。

### 5. UIパーツギャラリー（準備中）
汎用的なUIパーツを一覧できる専用エリアを構築済みです。現在は先行公開のため「Coming Soon」としていますが、内部的にはサイト本体の影響を受けずにパーツ単体を検証できる高度なプレビュー環境を実装しています。

## 開発の品質について

### あらゆる端末への最適化
パソコン、タブレット、スマートフォンなど、どのデバイスで見ても最適なレイアウトで表示されるレスポンシブ設計を徹底しています。また、ブラウザの文字サイズ設定を尊重する単位指定を行うなど、アクセシビリティにも配慮しています。

## 使用技術
- 開発フレームワーク：React 19 / Next.js 16
- 言語：TypeScript
- スタイリング：Tailwind CSS（tailwind-variants） / SCSS Modules
- アニメーション：GSAP
- フォーム・メール送信：Server Actions / Resend / Cloudflare Turnstile / zod
- 状態管理（Global）：zustand

## セキュリティと運用自動化
新しい制作実績やスキルの更新は、専用のデータファイルを編集するだけでサイト全体に自動反映される仕組みを構築しており、迅速な情報更新が可能です。
インデックス制御: middleware.ts による Basic認証 を適用（開発フェーズ）。認証通過時にはレスポンスヘッダーへ x-robots-tag: noindex, nofollow を動的に注入し、検索エンジンへの意図しない露出を遮断します。
Developer Experience (DX):
- スクリプトによる、UI GALLERY用テンプレートとメタデータの自動生成。
- スクリプトによる、SVGのコンポーネント化とハブファイル登録の自動化。
